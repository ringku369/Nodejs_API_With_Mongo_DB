## Installation

```sh
git clone this-url
```

```sh
cd project-root
```

##### please install [nodejs](https://nodejs.org/en/download/) and [mongodb](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/) dependencies of this project by running

```sh
npm install
```

```sh
npm start
```
